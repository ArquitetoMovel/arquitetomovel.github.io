**PowerShell ®**.

O **PowerShell** venho para substituir o velho *cmd.exe*, trazendo uma linguagem de 
script com mais recursos para se trabalhar em um ambiente com Windows.

### Ambiente de desenvolvimento do PowerShell

O PowerShell possui uma IDE de desenvolvimento para testar seus scripts e módulos 
Inclusive com suporte a depuração.

![PowerShellISE](https://pybxew-ch3302.files.1drv.com/y3pzHTZQT83SEPgTeO8LqAE3oHXnXZGLXybnDSgxm2-B4CyE9rsm_xMOqPvNGW-HvyEMaCWmFa4NhnkK7UA4Sq6BG5KenOAHKvhmd7V9L4ojGvJID-AMxctqFdu733J30FpPs8XvRc4Fuv19evxw8SMVR9pL5IkOGhBU54lq3fLZoU/PowerShell_ISE.png?psid=1)

Um dos pontos fortes *Linux* é a flexibilidade e quantidade de recursos disponíveis 
para que um profissional possa customizar e automatizar seu ambiente via linha de 
comando. 

Pois a Microsoft fecha essa lacuna no **Windows** com o PowerShell.

Poucas profissionais como desenvolvedores e adminitradores Windows conhecem ou se 
quer já tiveram a curiosidade de abrir aquele *prompt de comando Azul*. 

Pois bem, eu quero te ajudar a tirar o máximo dessa ferramenta. 

É possivél fazer muitas coisas com **Power Shell ®**. Mas vamos começar com o 
clássico _Olá Mundo_. 

Apenas acesse o menu iniciar do Windows, localize e abra o **PowerShell** e digite: 

 ` Write-Host "Olá Mundo"  `
 
 Após o ` [Enter]` veremos um saudoso *Olá Mundo* impresso no prompt.
 
 No proximo artigo, veremos como salvar nosso programa **PowerShell** e rodar o 
 mesmo diversas vezes.
